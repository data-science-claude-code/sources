# 자연어 SQL 프롬프트

세 개의 CSV 파일이 있습니다: customers.csv, orders.csv, products.csv. 이 파일들을 인메모리 SQLite 데이터베이스에 불러온 후, 아래 질문에 대한 SQL 쿼리를 작성하세요.

1. "완료된 주문만을 대상으로, 고객 등급별 총 매출을 보여주세요"
2. "최근 6개월 동안 가장 많은 금액을 지출한 고객 10명은 누구인가요?"
3. "제품 카테고리별 전월 대비 매출 성장률은 어떻게 되나요?"
4. "3건 이상 주문했지만 환불이 한 번도 없는 고객을 찾아주세요"
5. "지역별 평균 주문 금액은 얼마이며, 전체 평균과 비교하면 어떤가요?"

각 쿼리에 대해 SQL을 보여주고, 실행한 뒤 결과를 출력하세요.
각 결과에 대한 시각화를 하고 저장해줘
