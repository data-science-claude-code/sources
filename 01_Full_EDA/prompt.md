# 전체 EDA 프롬프트

ecommerce_sales.csv라는 이커머스 판매 데이터 셋이 있습니다. 이 데이터 셋에 대해 탐색적 데이터 분석(EDA)을 수행해 주세요. 다음 항목들을 확인하고 싶습니다:
- 데이터셋 먼저 살펴보기(pandas dataframe으로 전체 데이터 출력)
- 데이터셋의 shape, 데이터 타입(dtypes), 결측값 요약
- 모든 수치형 열에 대한 기술 통계량
- revenue(매출)와 quantity(수량)의 분포 플롯
- 모든 수치형 열에 대한 상관관계 히트맵
- 총 매출 기준 상위 5개 상품 및 카테고리
- 지역별 매출 분석
- 발견된 이상값 또는 데이터 품질 문제 표시

모든 차트는 PNG 파일로 저장하고, 주요 발견 사항에 대한 서면 요약을 제공해 주세요.
